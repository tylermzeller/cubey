Author: Tyler Zeller

A fully functional puzzle cube written in Python 2.7.10.

# Download

Download with:

```
pip install cubey
```

# Usage

Within a python interpreter type the following:

```Python
import cubey
c = cubey.Cube()
print c
```

which should print:

```Bash
 F   B   L   R   U   D
RRR PPP GGG BBB WWW YYY 
RRR PPP GGG BBB WWW YYY 
RRR PPP GGG BBB WWW YYY 
```
